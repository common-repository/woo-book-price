Woocommerce Book Price is a powerful, extendable WooCommerce plugin that helps you quick change product price without update one by one.

== Description ==

Woocommerce Book Price is a free plugin that allows you bulk update price of product without update one by one. Built to integrate seamlessly with WordPress and Woocommerce, Woocommerce Book Price provide feature:
- Create / Update / Delete Price book list.
- Base on book start date and end date , check the product and increase suitable price.
- No effect to original price of product.
- Simple , easy manage by csv import,
- Working good with plugin [Openpos - Point of sale plugin for woocommerce](https://codecanyon.net/item/openpos-a-complete-pos-plugins-for-woocomerce/22613341 "Openpos - Point of sale plugin for woocommerce")  with multi outlets

Here's a link to [Demo](https://plugins.openswatch.com/premium-openpos/wordpress/intro/ "Openpos - Price book plugin") 


== Installation ==

= Minimum Requirements =

* PHP version 5.6 or greater
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* WordPress 4.1+
* WooCommerce 3.3+



= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Woocommerce Book Price, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Woocommerce Book Price” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our Woocommerce Book Price plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
